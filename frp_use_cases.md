# FRP Use Case Scenarios

## Core Scenarios

### 1. Remote Coding (VS Code / SSH)
- Work on your home PC from anywhere
- Use VS Code Remote SSH
- Your desktop becomes your cloud dev machine

---

### 2. SSH into Your Local Machine
- Access your Windows PC remotely
- Run scripts, debug, restart services

---

### 3. Expose Local Servers to the Internet
- Share local Node/NestJS apps with clients or teammates
- No need to deploy for previews

---

## Developer / Engineering Use Cases

### 4. Webhook Testing
- Receive webhooks locally from:
  - Stripe
  - Paystack
  - GitHub

---

### 5. Mobile App Backend Testing
- Android/iOS apps can hit your local API
- No need to deploy backend

---

### 6. Demo Environments
- Show features live from your machine
- Fast iteration without CI/CD delays

---

### 7. Access Internal Tools
- Expose admin dashboards or APIs
- Useful for debugging and internal access

---

### 8. Team Collaboration
- Share your running environment with teammates
- Quick feature validation

---

## DevOps / Infrastructure Use Cases

### 9. CI/CD Callback Endpoints
- Tools like TeamCity can reach your local machine

---

### 10. Temporary Staging Environment
- Use local machine as staging
- Great for QA testing

---

### 11. Debug Production-like Issues
- Replay traffic from VPS to local app
- Simulate real-world scenarios

---

## Personal / Power User Use Cases

### 12. Remote Desktop Access
- Access your PC via RDP or VNC

---

### 13. File Access / Personal NAS
- Access files remotely

---

### 14. Self-hosting Experiments
- Run personal tools or dashboards

## Advanced Scenarios

### 15. Multi-service Routing
- app.domain.com → frontend

- ci.domain.com → CI tools

---

### 16. Replace ngrok
- Persistent tunnels
- No limits

---

### 17. Hybrid Cloud Setup
- VPS = public entry
- Local machine = compute engine
